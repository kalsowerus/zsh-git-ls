# Install (Antigen)

```zsh
antigen bundle kalsowerus/zsh-git-ls
```

# Usage

Introduces a new command `git-ls`, that works just like `ls -l`,
but shows the status of each file when listing a git directory.

## Supported options

```zsh
-a, --all
-A, --almost-all
    --group-directories-first
-h, --human-readable
-r, --reverse
-S
-t
```
